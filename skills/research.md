# Skill: Research

## Description
Use when raw material needs to be turned into evidence-backed notes, contradictions, open questions, and confidence tags.

## Required inputs
- relevant task file
- raw source files or URLs
- `MISSION.md` if the research must be mission-aligned

## Process
1. Read the sources closely.
2. Extract the most relevant facts.
3. Separate fact from inference.
4. Record contradictions explicitly.
5. Record open questions explicitly.
6. Assign confidence levels.

## Output format
```md
# Research Note

## Source summary
...

## Key facts
- ...

## Inferences
- ...

## Contradictions
- ...

## Open questions
- ...

## Confidence
- High:
- Medium:
- Low:
```

## Fail conditions
- unsupported claims
- missing source references where expected
- contradictions silently ignored
