# Skill: Synthesis

## Description
Use when several research notes or derived sources need to be turned into a structured understanding for downstream planning or execution.

## Required inputs
- at least two relevant source or derived files
- the task file
- `MISSION.md` if synthesis must align to mission

## Process
1. Identify the common thread across the inputs.
2. Preserve distinctions and contradictions.
3. Group findings into themes or decisions.
4. State what is known, inferred, and unknown.
5. Produce a concise downstream-ready summary.

## Output format
```md
# Synthesis Note

## Core theme
...

## What is known
- ...

## What is inferred
- ...

## What remains unknown
- ...

## Contradictions
- ...

## Recommended downstream use
- ...
```
