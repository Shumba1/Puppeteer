# Skill: Dashboard Update

## Description
Use when the dashboard view needs to be refreshed from repo truth.

## Required inputs
- `STATE.json`
- `RISKS.json`
- task folders
- output folders
- approvals feed
- evidence and log files

## Process
1. Read canonical state.
2. Recompute counts and health.
3. Write machine-readable dashboard feeds.
4. Never invent state.
